先上图

List

有序可重复，可以排序，有三种遍历方式

  1. 通过下标和size（）来遍历
  2. 用迭代器遍历
  3. 用forEach

ArrayList：

底层是动态数组,这个数组先有一个初始大小，如果list中添加的元素超过这个大小后，就新创建一个数组，大小是这个数组的二倍，然后把原来数组中的元素复制进来。由于底层是数组，查询快，在中间增加和删除慢

LinkedList：

底层是一个双向链表，查询慢，增删快，因为查询需要一个节点一个节点的找。在中间插入和删除快。删除的话只需要

Set

无序不可重复 （输入顺序和输出顺序不一致） 有两种遍历方式

  1. 迭代器
  2. forEach

HashSet （无序不可重复）

底层是HashMap，哈希表结构依赖于hashCode（）和equals（）方法，保证唯一性是通过先判断hashCode（）是否相同，如果相同的话判断equals（）返回的是否为true

TreeSet

底层是TreeMap，可以排序，分为 自然排序和比较器排序

  1. 自然排序：元素具备比较性，让元素所属的类实现Comparable接口
  2. 比较器排序：集合具备比较性，给TreeSet的构造中传一个比较器接口的子类对象Comparator
    唯一性是根据comparator方法的返回值是否为0来决定的

    LinkedHashSet

    底层依赖哈希表和链表。哈希表保证元素的唯一性，链表保证元素有序

Map （无序 key不可重复）

转换成keySet和EntrySet来遍历

HashMap

是基于哈希表的Map实现，哈希表保证键的唯一性。

LinkedHashMap

是Map接口的哈希表和链表的实现，有序。由哈希表保证键的唯一性，链表保证有序。

TreeMap

是基于红黑树的Map接口实现。
